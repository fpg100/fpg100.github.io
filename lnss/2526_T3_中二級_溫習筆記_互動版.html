三公共財政與市民的關係
① 市民對公共財政的權利和責任
1.1 享有社會福利的權利

市民可依法享有政府提供的福利和免費或廉宜的公共服務，如12年免費教育、公共醫療服務、文娛康樂設施等，以滿足基本需要。

1.2 履行納稅的責任

政府開支主要來自市民的稅款，市民有義務遵守《稅務條例》，按時報稅及繳稅。

② 市民參與公共財政政策制定的正確態度

市民有責任以積極、理性的態度參與政策制定，讓政府更合理地分配公共資源，平衡各方利益。

2.1 理性分析公共財政政策

先了解不同持份者的意見，從社會整體福祉的角度全面分析。

2.2 以恰當渠道表達意見

政府每年進行公眾諮詢，市民可透過恰當渠道提出意見；政府官員亦會出席諮詢會，在場人士可發表意見。

一量度經濟表現的指標
A 本地生產總值（GDP）

一段時間內生產的貨品和服務的市場總值。數值越高，代表生產活動越活躍。

香港近年 GDP 呈 上升 趨勢，整體經濟表現穩步增長
B 人均本地生產總值

平均每人所佔的 GDP。越高代表當地居民生活水平越高。

人均 GDP ＝ GDP ÷ 人口總和

香港近年人均 GDP 呈 上升 趨勢

C 失業率

失業人口在勞動人口中所佔的百分比。

失業率 ＝ 失業人口 ÷ 勞動人口 × 100%
  • 經濟理想 → 企業擴充業務，失業率低
  • 經濟欠佳 → 企業裁減人手，失業率高
  • 1997–2023年最高：2003年 7.9%，平均約 4.4%
D 通脹率

整體物價水平持續變動的比率。

0%以上 → 通貨膨脹 0%以下 → 通貨緊縮
  • 過高 → 生活成本上升
  • 過低 → 打擊企業投資意欲
  • 1997–2023年平均 1.5%；1999–2004年曾現通縮
1.2產業結構的分類
第一產業

從事種植、養殖或開採自然資源，例如農業、漁業。

香港早已式微，2020年僅佔GDP 0.1%

第二產業

把天然或人工原料加工的生產活動，例如製造業、建造業、能源業。

1980年代起日漸萎縮

第三產業

提供服務的生產活動，例如零售業、金融業、旅遊業。

目前佔GDP超過九成，繼續茁壯成長

第三產業的發展

四個主要行業：金融服務 ／ 旅遊 ／ 貿易及物流 ／ 專業及其他工商業支援服務

六項優勢產業：文化及創意 ／ 醫療 ／ 教育 ／ 創新科技 ／ 檢測及認證 ／ 環保

②香港經濟的特徵
2.1 高度外向和開放型的經濟
A 高度外向型

對外貿易總額一直是GDP的2倍以上。內地和海外經濟繁榮，香港便受惠；反之則受打擊。

B 高度開放型

大部分貨物、人員、資金、資訊可自由進出香港，政府甚少干預或補貼經濟活動。

六大自由
貿易自由 營運自由 資金進出自由 國際航運自由 出入境和逗留自由 資訊自由
2.2 地區或全球事件與香港經濟的關係
金融貿易危機 ＆ 疫情肆虐
1997亞洲金融風暴：貨幣匯率遭衝擊，股市、樓市下挫，企業嚴重虧損
2003沙士疫情爆發，政府實施公共管制措施，旅客無法自由來港
2008全球金融危機爆發，香港經濟受打擊
2018中美貿易摩擦，各自向對方貨物加徵關稅或實施貿易限制
2020新冠疫情全球爆發，窒礙香港經濟發展
2.3 香港經濟安全對國家的重要性
經濟安全定義

消除和化解潛在風險，抗拒外來衝擊，確保國家經濟利益持續穩定發展；是國家富強的有力保障。

貿易方面

香港扮演外資與內地企業的中介人角色，加強聯繫內地和國際產業鏈。

金融方面

人民幣國際化進程推進，「滬港通」、「深港通」等落實，在引入外資上發揮巨大作用。

③香港經濟與國家發展的緊密關係
3.1 二戰後香港經濟發展概略
A 奠定工業基礎

二戰後國共內戰，大批內地民眾湧港，為香港帶來資金、技術和勞動力。

B 製造業興起

1950年韓戰後聯合國對華禁運，紡織、玩具、電子等製造業迅速興起。1960–70年代是全球重要製造業基地。

C 服務業興起

1978年國家實行改革開放，製造業北移，香港轉型為以金融、航運、貿易為核心的服務業，成為國際金融中心。

3.2 改革開放帶來的機遇和香港的貢獻
機遇
  • 產業轉型：「前店後廠」模式，廠家遷往珠江三角洲
  • 經貿地位提升：對香港金融服務需求不斷增加，成國際金融、航運和貿易中心
香港對改革開放的貢獻
  • 引入外資：香港商人率先投資，提升外企信心
  • 示範作用：借鏡香港管理策略，完善社會主義市場經濟
3.3 內地與香港越趨緊密的經濟聯繫
A 貿易量增加

內地是香港最大的貿易伙伴；2022年佔香港貿易總額約一半。

B 人才交流頻繁

「高端人才通行證計劃」；「十四五規劃」鼓勵港人到大灣區創業就業。

C 各類投資活動增加

內地是香港第二大外來直接投資來源地；兩地股票市場互聯互通。

香港擔當「超級聯繫人」：具健全法制，連接內地經濟與全球市場；支持開辦離岸人民幣業務，建立互聯互通機制。

二經濟全球化為香港帶來的機遇和挑戰
1.1 經濟全球化的含義

跨越國界的經濟活動日趨頻繁，原料、勞動力、商品、服務、資本等經濟要素在全球不斷流動，各經濟體互相影響，漸漸趨向一體化的現象。

五大特徵（市場一體化）
原料市場 金融市場 生產鏈 商品和服務市場 勞工市場
1.2 促進經濟全球化的因素
A 各地政府推動

提倡自由貿易，組建區域性經濟組織，消除關稅和貿易壁壘。

B 交通運輸進步

提高運輸速度和運載量，降低成本，刺激國際經貿交流和合作。

C 資訊科技發達

通訊和網絡技術突破，商戶藉互聯網進行跨地域的經濟活動。

D 跨國企業推動

已發展國家的跨國企業在各地採購、生產、銷售，令各地經濟緊密連在一起。

② 經濟全球化的影響
2.1 促進經濟增長

發展中國家設立代工廠，創造就業，改善居民收入和生活水平。

2.2 造成經濟波動

各地生產、銷售、投資互相依賴，一國不景可拖累他國。

2.3 生產互相依賴

一地生產停頓，容易導致全球生產鏈中斷。

2.4 改變產業發展

跨國企業到發展中國家設廠引入技術，惟本地企業或不敵外來競爭被迫結業。

2.5 增加消費選擇

消費者享有更多選擇，但跨國企業或壟斷本地市場。

2.6 改變勞工待遇

把低技術工序外判，廠商或削減工資，損害勞工權益；已發展國家工人或因工廠外移而失業。

③香港作為國際金融中心
3.1 國際金融中心的特徵
A 金融市場成熟

匯聚知名企業和金融機構，頻繁進行融資、放貸等活動。

B 金融服務多元

股票、債券、基金、保險、資產管理等多元化服務。

C 健全監管和法律制度

保障投資者利益，須有健全的金融監管制度及公正法律制度。

D 資金自由流動

大量資金每天從各地流入，然後自由地轉移到其他地方。

E 市場資訊自由流通

投資者根據即時市場消息，及時作出投資決定。

F 奉行市場經濟

投資活動由投資者自行決策和自負盈虧，政府不會隨便干預。

3.2 香港成為國際金融中心的原因
A 地理位置優越
  • 時區在時差上成為歐美橋樑，形成接近24小時環球金融市場
  • 毗連珠三角，是中國重要出口貿易基地
B 法治基礎深厚
  • 司法獨立，保障私有財產
  • 完善金融監管制度和法例
C 稅制簡單及稅率低

稅率偏低，不設資本增值稅及遺產稅，吸引內外企業和投資者。

D 背靠內地，面向世界

內地企業需會計、財務等服務；外資企業透過香港進入內地投資。

F 奉行自由經濟

不設外匯管制，港元與美元聯繫匯率掛勾；無官方規定的外幣匯率。

G 專業人才充裕

注重「兩文三語」，具備溝通能力；多間大專院校開辦金融課程；透過專才輸入計劃招募環球人才。

3.3 機遇及挑戰
機遇
  • 突顯中外橋樑角色：吸引大量跨國企業和投資者
  • 服務素質提升：吸引頂尖金融人才和先進技術
  • 促進交流合作：學習先進金融管理經驗，與他中心互動
挑戰
  • 金融市場波動性增加，須防控動盪
  • 面對其他國際金融中心的競爭
  • 金融科技和虛擬金融產品快速發展，須更新監管制度
  • 全球對高端金融人才需求增加
④ 香港在國家經濟發展的角色
4.1 先行者

成為全球最大離岸人民幣中心，為內地金融業國際化發揮防火牆、試驗田功能。

4.2 聯繫者

「一帶一路」超級聯繫人，協助內企「走出去」；實行普通法，成國際著名仲裁中心。

4.3 推廣者

與多國建立貿易伙伴關係；舉辦「一帶一路高峯會」，提供商貿和服務平台。

三香港經濟起伏對青年就業帶來的機遇和挑戰
① 香港近年的就業情況
三級產業就業人數
  • 1970年代起，重心由第二產業轉移到第三產業
  • 2000–2022年：第二產業由16%下跌至近12%；第三產業由83%上升至約88%
青年整體失業率
  • 近年青年就業較整體人口差
  • 15–19歲失業率較高
  • 20–29歲比總體失業率高
青年就業分布
  • 約90%的15–39歲青年投身第三產業
  • 從事「公共行政」和「金融」服務的青年就業率逐漸上升
  • 六項優勢產業中以文化及創意產業比例最高
② 青年就業的機遇與挑戰
2.1 經濟起伏

香港高度外向，易受環球影響。經濟不景 → 失業率↑，青年就業困難；經濟好轉 → 就業機會增加。

2.2 新舊行業的興替

部分重複性強的傳統職位被人工智能取代；新興行業如程式編寫員和數據分析師湧現。

2.3 就業模式新趨勢：零工經濟

青年以自僱形式，通過手機或互聯網平台接洽工作，成為「斜槓族」，合作方式和工作時間由自己洽商。

好處
  • 就業有更大自由度
  • 自行安排工作內容和時間
  • 累積經驗，拓展人際網絡
弊處
  • 工作具高重複性，難以提升技能
  • 缺乏福利保障和勞工權益
  • 規劃不善易耽擱工作進度
2.4大灣區為香港青年帶來的機遇與挑戰

由香港、澳門及廣東省九個城市組成「粵港澳大灣區」；2019年公布《粵港澳大灣區發展規劃綱要》，全面規劃大灣區的發展目標和範疇。

機遇
  • 交通完善：形成「一小時生活圈」，提供廣闊就業和創業空間
  • 產業優勢：科技創新領域具優勢，可參與創業活動
  • 政府協助：設立「大灣區青年就業計劃」及創新創業孵化基地
挑戰
  • 文化差異：香港與內地社會習慣不同，需時適應
  • 人才競爭：需與全國各地優秀人才競爭
  • 法制差異：香港與內地的法律體系不同，需了解適應
